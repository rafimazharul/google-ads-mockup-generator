# Google Ads Mockup Generator

A modern, responsive tool for generating and previewing Google Search Ads mockups in both Desktop and Mobile views.

## Features

- **Live Preview**: See changes in real-time as you type.
- **Dark Mode Support**: Preview how ads look in Google's official dark mode aesthetic.
- **Responsive Design**: Fits perfectly in the viewport with a fixed footer and independent scrollable panels.
- **Desktop & Mobile Views**: Toggle between different device previews.
- **Copy & Download**: Easily copy the ad text or download the mockup as a PNG image.

## Technology Stack

- **React**: For the user interface and state management.
- **Babel**: To compile JSX in the browser.
- **html2canvas**: For generating downloadable PNG images.
- **CSS3 Variables**: For theme-aware styling.

## How to Run

Simply open `index.html` in your web browser, or use a local development server like `npx serve .`.
